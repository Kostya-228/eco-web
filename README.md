# EcoHub Frontoffice
Веб морда для приложения EcoHub. Общается с открытыми методами через
админску апи (api_admin).

## Запустить локально
1) Скачать зависимости
    ```
    npm i
    ```
2) Запустить
    ```
    npm run serve
    ```

## Запустить на сервере
1) Собрать образ
    ```
    docker-copmose build
    ```
2) Запустить
    ```
    docker-compose up -d
    ```
